# Light & Dark Mode Toggle Project

A simple and interactive **Light/Dark Mode Switcher** built using **HTML, CSS, and JavaScript**.  
This project demonstrates DOM manipulation, event handling, conditional rendering, and smooth UI transitions.

---

## Features
- Smooth Light/Dark mode transition  
- Gradient button design  
- Animated hover effects  
- Responsive heading styling  
- Clean and minimal UI  
- Beginner-friendly JavaScript logic  

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)

---

## How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Simply open **index.html** in any browser
4. No installation or server required
